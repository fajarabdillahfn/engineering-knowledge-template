# Knowledge

Engineering knowledge base.

This directory holds the canonical engineering knowledge for the project: architecture, glossary, conventions, service references, flow descriptions, troubleshooting guides, operational playbooks, and architectural decision records.

## Structure
- `architecture.md` — System-level architecture overview
- `glossary.md` — Terms, acronyms, and definitions
- `conventions.md` — Engineering standards and conventions
- `services/` — Per-service documentation
- `flows/` — End-to-end flow documentation
- `troubleshooting/` — Issue and incident references
- `playbooks/` — Operational runbooks
- `decisions/` — Architectural Decision Records (ADRs)
